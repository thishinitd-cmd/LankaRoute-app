# 🗺️ LankaRoute - Real-time Transit Tracking System

LankaRoute is a robust mobile solution designed to tackle the unpredictability of public transportation in Sri Lanka. By leveraging real-time geolocation data, this app empowers commuters to track buses and trains live on a map, ensuring they never miss a ride again.

---

## 🌟 Key Features

* **Live Transit Tracking:** View the real-time location of buses and trains on an interactive map.
* **Smart Proximity Alerts:** Identify the nearest bus stops and railway stations based on your current GPS location.
* **Route Search:** Search for specific transit routes and see the active vehicles currently operating on those paths.
* **UI/UX Focused Design:** A clean, minimalist interface designed in Figma to provide a seamless user experience.

---

## 🛠️ Tech Stack

**Frontend:**
* React Native (Cross-platform Android/iOS)
* React Navigation (Fluent screen transitions)
* Context API (Global state management)

**Backend:**
* Node.js & Express.js (Scalable RESTful API)
* MongoDB (Flexible NoSQL data storage)

**APIs & Tools:**
* Google Maps SDK & Geolocation API
* Figma (UI/UX Prototyping)
* Git/GitHub (Version Control)
